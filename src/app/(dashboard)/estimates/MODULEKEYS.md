# Module Keys — Estimativas

Os seguintes moduleKeys devem ser configurados em /access para os perfis:

| moduleKey | Perfil | Visibilidade |
|---|---|---|
| estimates_effort | CONSULTANT, MANAGER, ADM_SIST | Horas e esforço (sem preços) |
| estimates_financial | MANAGER, ADM_SIST | Preços, contingência e desconto |

Configure via: /access → selecione o perfil → adicione o moduleKey
